# MueblesCraft - Landing Page para Fabricación de Muebles

Una landing page moderna y profesional para un negocio de fabricación artesanal de muebles.

## 🌟 Características

### ✨ Diseño Moderno
- Diseño responsive que se adapta a todos los dispositivos
- Interfaz limpia y profesional con esquema de colores de madera
- Animaciones suaves y transiciones elegantes
- Tipografía moderna con Google Fonts (Inter)

### 📱 Funcionalidades Principales
- **Navegación fija** con efecto glassmorphism
- **Menú hamburguesa** para dispositivos móviles
- **Sección Hero** con llamadas a la acción prominentes
- **Galería interactiva** con modal de vista ampliada
- **Botón de WhatsApp flotante** con animación pulse
- **Formulario de contacto** que redirige a WhatsApp
- **Secciones de servicios** con iconos y efectos hover

### 🎨 Secciones Incluidas
1. **Header/Navegación** - Menú fijo con enlaces suaves
2. **Hero Section** - Presentación principal con botones CTA
3. **Servicios** - 6 servicios principales de fabricación
4. **Galería** - Portafolio de trabajos realizados
5. **Contacto** - Información de contacto y formulario
6. **Footer** - Enlaces adicionales y redes sociales

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura semántica y accesible
- **CSS3** - Diseño moderno con Flexbox y Grid
- **JavaScript ES6+** - Interactividad y funcionalidades dinámicas
- **Font Awesome** - Iconografía profesional
- **Google Fonts** - Tipografía moderna
- **Imágenes de Unsplash** - Fotografías de alta calidad

## 🛠️ Instalación y Uso

1. **Clona o descarga** los archivos del proyecto
2. **Abre** `index.html` en tu navegador web
3. **Personaliza** el contenido según tus necesidades

### 📝 Personalización

#### Información de Contacto
Edita estas secciones en `index.html`:
- Número de teléfono en el botón de WhatsApp (línea 234)
- Información de contacto en la sección de contacto
- Enlaces de redes sociales en el footer

#### Contenido
- **Servicios**: Modifica los servicios en la sección `#servicios`
- **Galería**: Reemplaza las imágenes con tus propios trabajos
- **Información de la empresa**: Actualiza nombre, descripción y datos

#### Estilo
- **Colores**: Cambia la variable `--primary-color` en CSS
- **Tipografía**: Modifica las fuentes en la importación de Google Fonts
- **Imágenes**: Reemplaza las URLs de Unsplash con tus propias imágenes

## 🔧 Configuración de WhatsApp

Para usar el botón de WhatsApp:

1. Reemplaza `1234567890` con tu número real (incluye código de país)
2. El formato debe ser: `521234567890` (ejemplo para México)
3. El mensaje se puede personalizar en el archivo JavaScript
